# Education Insights Agent (ADK Demo)

This project is an educational analytics agent built using Google's Agent Development Kit (ADK).
The agent analyzes Azerbaijani school graduate performance data (1995–2023) and can answer a wide range of analytical questions, including:

* Yearly school statistics

* Acceptance rankings

* Male vs. female performance comparison

* Long-term trends (1995–2023)

* Regional summaries

* Anomaly detection

* School-to-school comparisons

Dataset used : https://huggingface.co/datasets/nijatzeynalov/az-school-graduate-enrollment
